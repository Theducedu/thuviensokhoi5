Hướng dẫn lấy API Key Groq

  1. Truy cập trang web: console.groq.com/keys
  2. Đăng nhập bằng tài khoản Google (hoặc tạo tài khoản mới).
  3. Nhấn vào nút Create API Key. Nhập tên tùy ý (VD: nhanxettt27) rồi Submit.
  4. Sao chép toàn bộ dãy mã có dạng gsk_... vừa xuất hiện trên màn hình.
  5. Quay lại ứng dụng này, dán dãy mã vừa chép vào ô "Groq API Key" và nhấn Kiểm tra.

💡 Mã API Key từ Groq hiện đang được cung cấp hoàn toàn miễn phí. Tốc độ phản hồi cực nhanh, phù hợp cho việc tự động điền hàng trăm nhận xét trong vài giây.
